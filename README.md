# contador-de-cliques

Uma aplicação React simples que conta quantas vezes o botão foi clicado. Este projeto foi criado para praticar componentes funcionais, gerenciamento de estado com `useState` e eventos de clique no React.

## Instalação e uso

1. Abra um terminal e clone o repositório:

```bash
git clone https://github.com/usuario/contador-de-cliques.git
```

2. Entre na pasta do projeto:

```bash
cd contador-de-cliques
```

3. Instale as dependências:

```bash
npm install
```

4. Inicie o servidor de desenvolvimento:

```bash
npm start
```

5. Abra o navegador e acesse:

```text
http://localhost:3000
```

A página vai mostrar o contador e o botão. Clique no botão para incrementar o valor.
